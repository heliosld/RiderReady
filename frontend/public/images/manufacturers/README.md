# Manufacturer Logos

Place manufacturer logo files here using the naming convention: `[slug].png` or `[slug].svg`

## Required Logos:
- robe.png (Robe Lighting)
- martin.png (Martin Professional)
- clay-paky.png (Clay Paky)
- chauvet.png (Chauvet Professional)
- glp.png (GLP)
- ayrton.png (Ayrton)
- etc.png (ETC)

## Guidelines:
- Use transparent backgrounds (PNG or SVG preferred)
- Recommended size: 400x200px (2:1 aspect ratio)
- File size: Keep under 100KB per logo
- Format: PNG or SVG for best quality
